# 🧬 Weave Landing Page

Premium marketing landing page for Weave Protocol — your yield, automated, forever.

## ✨ Features
- **Apple-style Storytelling**: Scroll-triggered reveals and massive typography.
- **Interactive Yield Calculator**: Real-time comparison between simple APR and Weave's daily auto-compounding.
- **Institutional Branding**: Pure black aesthetic with high-fidelity purple glows and Space Grotesk/JetBrains Mono typography.
- **Mobile Optimized**: Responsive design for all device types.

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
